CARRIER LOGOS
=============
Drop official carrier logo files into this folder and the website's
trust bar will display them automatically. Until a file exists, the
site shows the carrier's name as a styled text badge instead.

Use these exact file names (PNG with transparent background, roughly
200-400px wide works best):

  canada-life.png      Canada Life
  rbc-insurance.png    RBC Insurance
  manulife.png         Manulife
  mackenzie.png        Mackenzie Investments
  fidelity.png         Fidelity
  quadrus.png          Quadrus Investments
  ipc.png              IPC Wealth

WHERE TO GET THEM: each carrier's advisor/marketing portal provides
approved logo packs for contracted advisors. Using those (rather than
logos found on the internet) keeps you onside with each carrier's
brand-use guidelines.

After adding files here, upload this carriers folder to your GitHub
repository the same way you upload the website files.
