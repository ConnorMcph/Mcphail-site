CARRIER LOGOS
=============
This folder already contains five carrier logos (SVG format) sourced
from public references, and the website displays them automatically:

  canada-life.svg      Canada Life
  rbc-insurance.svg    RBC Insurance
  manulife.svg         Manulife
  mackenzie.svg        Mackenzie Investments
  fidelity.svg         Fidelity

Two carriers had no public logo available, so they appear as styled
text badges on the site until you add files for them:

  quadrus.png          Quadrus Investments
  ipc.png              Investment Planning Counsel
                       (rename to IPC Wealth after the merger)

UPGRADING TO CARRIER-APPROVED LOGOS (recommended when convenient):
each carrier's advisor/marketing portal provides approved logo packs
for contracted advisors. Save them here as PNG files using the names
below — a PNG automatically takes priority over the bundled SVG with
the same name, so you never need to delete anything:

  canada-life.png, rbc-insurance.png, manulife.png,
  mackenzie.png, fidelity.png, quadrus.png, ipc.png

(Transparent background, roughly 200-400px wide works best.)

After adding files here, upload this carriers folder to your GitHub
repository the same way you upload the website files (drag the whole
folder into the upload area — the choose-files button can't select
folders).
